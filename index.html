Boggle Generator

This generates a 4x4 grid of random letters for playing Boggle. The function to generate in generate().

import random
letters=['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j', 'k', 'l', 'm', 'n', 'o', 'p', 'q', 'r', 's', 't', 'u', 'v', 'w', 'x', 'y', 'z']
def generate():
	for x in range(0, 4):
		print(random.choice(letters), random.choice(letters), random.choice(letters), random.choice(letters))
